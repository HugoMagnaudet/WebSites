Sur Windows
1- Lancer Xampp ,démarrer process apache + mysql
2- Déplacer le répertoire codeigniter dans c:\xampp\htdocs\
3- renommer le répertoire "codeigniter" en "isaq"
4 importer le schéma de la base mysql "isaq_db_test.sql" sous c:\codeigniter


Sur linux 
1- Déplacer le répertoire codeigniter dans le répertoire du serveur apache
2- renommer le répertoire "codeigniter" en "isaq"
3- Exécuter le fichier "isaq_db_test.sql" pour importer le schéma et les donées de la base 



Entrer 
http://localhost/isaq
dans le navigateur
